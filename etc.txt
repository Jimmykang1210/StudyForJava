1. 반드시 포함될 단어/문장을 지정
 - 검색어 처음과 끝에 큰 따옴표(")를 입력 해당 언어나 문장이 반드시 포함된 사이트만 검색 가능
2. 제외할 검색어를 지정
 - 검색어 앞에 마이너스(-)를 입력, 해당 검색어를 제외한 결과를 표시
3. 유의어 검색
 - 검색어 앞에 물결표시(~)를 입력, 검색어와 비슷한 의미를 가진 자료 표시
4. 단어의 정의 검색
 - 검색어 앞에 (define:단어)를 입력, 해당 검색어의 정의 표시
5. 특정 단어를 반드시 포함하여 제목 검색
 - 검색어에 (intitle:단어')를, 입력 제목에 반드시 특정 단어를 포함한 검색 결과 표시
6. 불확실한 검색어를 입력
 - 문장 사이에 별표(*)를 입력, 정확한 검색어가 떠오르지 않을 때 빈자리를 채워서 결과 표시
7. 숫자의 범위 지정
 - 검색할 숫자 사이에 마침표 2개(..)를 입력, 숫자 범위 내의 검색 결과 표시
8. 두 단어 중 아무거나 검색
 - 검색하고 싶은 두단어 사이에 (or)를 입력, 두 단어 중 둘 중 하나라도 있는 검색 결과
9. 특정 사이트 이내에서만 검색
 - 검색어에 (site:주소)를 입력, 해당 주소를 가진 사이트 내에서 해당 단어를 포함한 자료만 검색
10. 특정 사이트와 관계된 사이트 검색
 - 검색어에 (rel:주소)를 입력, 특정 사이트와 관련있는 사이트 검색 
11. 특정 사이트를 링크한 모든 사이트 검색 
 - 검색어에 (link:주소)를 입력, 특정 사이트를 링크한 모든 사이트를 검색
12. 특정 종류의 파일만 검색
 - 검색어에 (filetype:확장자명)를 입력, 찾고자 하는 것을 입력하고 끝에 검색
